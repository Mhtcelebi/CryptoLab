# 🛰️ CryptoLab v1.0

CryptoLab; siber güvenlik uzmanları, CTF oyuncuları ve yazılım geliştiriciler için tasarlanmış, tamamen komut satırı (CLI) üzerinden çalışan profesyonel bir kriptoloji aracıdır. Ekstra hiçbir harici kütüphaneye ihtiyaç duymadan (tak-çalıştır) hem Windows (CMD/PowerShell) hem de Linux (Kali Linux) terminallerinde kararlı ve hatasız bir şekilde çalışır.

CryptoLab is a professional, production-grade Command Line Interface (CLI) toolkit designed for cybersecurity enthusiasts, CTF players, and software engineers. Operating with zero external dependencies (plug-and-play), it runs flawlessly across both Windows (CMD/PowerShell) and Linux (Kali Linux) environments.

---

## ✨ Özellikler / Features

* **🌍 Çoklu Dil Desteği (Dinamik i18n):** Türkçe ve İngilizce dil seçeneği.
* **🛡️ Hata Yönetimi (Crash-Proof):** Girdi hatalarında veya `Ctrl+C` kesmelerinde çökmeyen mimari.
* **📦 Sıfır Bağımlılık (Zero Dependencies):** Sadece Python standart kütüphaneleri kullanılır.

---

## ⚙️ Desteklenen Algoritmalar / Supported Algorithms

### 1. Klasik Kriptoloji / Classical Ciphers
* Caesar Cipher (Sezar)
* Vigenère Cipher
* Affine Cipher (Doğrusal)
* Atbash Cipher
* ROT13
* Rail Fence Cipher

### 2. Kodlama & Taban Dönüşümü / Encoding & Base Conversion
* Base16 (Hexadecimal)
* Base32
* Base64
* Base85 (Ascii85)
* Binary (İkilik Sistem)

### 3. Modern Hash Fonksiyonları / Modern Hash Functions
* MD5 & SHA-1
* SHA-2 Ailesi (SHA-224, SHA-256, SHA-384, SHA-512)
* SHA-3 Ailesi (SHA3-224, SHA3-256, SHA3-384, SHA3-512)
* *Hash Doğrulama (Hash Verification) desteği mevcuttur.*
